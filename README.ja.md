# GHAutoSave
<p align="center">
  <img src="AutoSave.png" width="128">
</p>

Grasshopper の定義ファイル（`.gh`）を一定間隔で自動保存するプラグインです。  
AutoSave は **キャンバスにコンポーネントを置かず**、**Display メニュー**から ON / OFF を切り替えます。

他の環境へ `.gh` ファイルを共有しても、依存関係エラーが発生しない設計です。

---

## 機能

- **Display → AutoSave** から ON / OFF 切り替え
- 自動保存間隔：**30秒固定**
- Grasshopper コンポーネント不要
- `.gh` ファイル内に依存情報を残さない
- 複数端末・他人とのファイル共有に安全
- 未保存ファイルに対するキャンバス上のリマインド表示

---

## 動作概要
<img src="docs/display-menu.png" width="400">

- **AutoSave が ON の場合**
  - 一度保存された `.gh` ファイルは 30 秒ごとに自動保存されます
  - 未保存の定義には、キャンバス上に保存を促すメッセージが表示されます
- **AutoSave が OFF の場合**
  - 一切の動作を行いません

Grasshopper の安全仕様に従い、初回保存前のファイルは自動保存されません。
